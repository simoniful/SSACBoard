# Board

## Description
- 기본적인 커뮤니티의 게시판 기능 구현
- 기간: 2021.12.25 ~ 2022.1.6
- 회원가입, 로그인, 비밀번호 변경
- 게시판 글 작성, 수정, 삭제
- 댓글 작성, 수정, 삭제
- Pagination, RefreshControl

---

## Skill
- MVVM
- Swift, UIKit, Code-Based UI, Autolayout, SnapKit
- RxSwift
- URLSession, Codable
